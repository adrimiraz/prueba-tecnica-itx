# prueba-tecnica-itx

Servicio de consulta de tarifas para productos por fecha, marca y producto.

> Prueba técnica desarrollada con Java 17, Spring Boot 3

---

##  Tecnologías utilizadas

- Java 17
- Spring Boot 3
- Spring Data JPA
- H2 Database (en memoria)
- JUnit 5 + Mockito
- OpenAPI / Swagger
